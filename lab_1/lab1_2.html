Гиперссылка (англ. hyperlink) — часть гипертекстового документа, ссылающаяся на другой элемент.

Как сделать ссылку

Для создания ссылки используется тег <a> с атрибутом href.

Атрибут target

_blank — новое окно, _self — текущее окно, _parent — фрейм-родитель, _top — полное окно.

Изображение в качестве ссылки

Изображение помещается между тегами <a> и </a>.

Ссылки внутри страницы

Якорь создается с помощью атрибута id, ссылка на него — через #.
